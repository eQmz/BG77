Uso_de_COPY_PARSE_STR

Copia una cadena que se encuentra entre dos caracteres desde un buffer origen a otro buffer de destino. El proposito de esta MACRO es copiar cadenas de informacion que debe ser parseada por delimitadores como '
', ' ', '=', ','.

Copia una cadena que se encuentra entre dos caracteres desde un buffer origen a otro buffer de destino. El proposito de esta MACRO es copiar cadenas de informacion que debe ser parseada por delimitadores como '
', ' ', '=', ','

Parámetros
dstBuffEs el buffer de destino.
dstBuffSizeEs el tamaño total de dstBuff. Sirve para evaluar el desbordamiento de memoria.
srcBuffEs el buffer origen.
chr1Es el caracter que indica desde donde se empezar a copiar. Se empieza a copiar una posicion despues de este caracter.
chr2Es el caracter hasta el cual se va a copiar. Este ultimo caracter se cambia a nulo '\0' para delimitar la cadena.

En este ejemplo se copia la cadena "125" proveniente de dataBuff a parseBuff

uint8_t dataBuff[] = "vol=125v,pot=12w";
uint8_t parseBuff[30] = {'\0'};
COPY_PARSE_STR(sizeof(parseBuff), parseBuff, dataBuff, '=', 'v');
